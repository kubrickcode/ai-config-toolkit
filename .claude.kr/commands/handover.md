# 대화 인수인계 요약

AI 에이전트 인수인계를 위한 종합적인 요약 생성: $ARGUMENTS

## 이 명령의 기능

이 명령은 다른 AI 에이전트가 작업을 원활하게 이어받을 수 있도록 구조화된 전체 대화의 상세한 마크다운 요약을 생성한다. 요약에 포함되는 내용:

- **컨텍스트와 배경**: 초기 문제 정의와 사용자 요구사항
- **완료된 작업**: 수행된 모든 작업의 상세 목록
- **기술적 결정**: 주요 아키텍처 및 구현 선택사항
- **코드 변경**: 수정, 생성 또는 리팩토링된 파일 요약
- **현재 상태**: 프로젝트의 현재 위치
- **대기 중인 항목**: 미완료 작업이나 향후 고려사항
- **중요 사항**: 계속 진행을 위한 핵심 정보

## 출력 형식

다음 구조의 마크다운 파일을 생성한다:

```markdown
# 프로젝트 인수인계 요약

## 생성일: [타임스탬프]

## 📋 개요

프로젝트와 주요 목표에 대한 간략한 설명

## 👤 사용자 컨텍스트

- 사용자의 기술적 배경과 선호도
- 구체적인 요구사항과 제약사항
- 커뮤니케이션 스타일 선호도

## 🎯 원래 요구사항

- 초기 문제 정의
- 주요 목표와 목적
- 성공 기준

## ✅ 완료된 작업

### 작업 1: [설명]

- 수행한 내용
- 이 접근법을 선택한 이유
- 주요 코드 변경사항
- 영향받은 파일

### 작업 2: [설명]

...

## 🏗️ 프로젝트 구조

코드베이스의 현재 상태:

- 디렉터리 구조
- 주요 파일과 그 용도
- 추가/수정된 의존성

## 🔧 기술적 결정

### 결정 1: [주제]

- 고려한 옵션
- 선택한 접근법
- 근거

### 결정 2: [주제]

...

## 💻 코드 예시

중요한 구현을 보여주는 핵심 코드 스니펫

## 🚧 현재 상태

- 작동 중인 기능
- 작업 중인 기능
- 알려진 이슈나 제한사항

## 📝 대기 중인 작업

- [ ] 작업 1
- [ ] 작업 2
- [ ] 작업 3

## ⚠️ 중요 경고사항

- 변경 사항 깨짐 방지를 위한 핵심 정보
- 보안 고려사항
- 성능 영향

## 🔄 다음 단계

작업 계속을 위한 권장 조치

## 🗂️ 관련 파일

- 파일 1: 용도와 최근 변경사항
- 파일 2: 용도와 최근 변경사항

## 💡 추가 컨텍스트

원활한 계속을 위한 기타 관련 정보
```

## 이 요약의 이점

- **연속성**: 새 에이전트가 정확히 중단된 지점부터 시작 가능하다
- **컨텍스트 보존**: 모든 중요한 결정과 근거가 문서화된다
- **효율성**: 정보 재설명이나 재발견 필요성이 감소한다
- **명확성**: 구조화된 형식으로 정보를 쉽게 찾을 수 있다
- **완전성**: 기술적 및 대화적 컨텍스트 모두를 캡처한다

## 사용 예시

기본 사용:

```
현재 대화의 인수인계 요약 생성
```

특정 초점:

```
인증 구현에 초점을 맞춘 인수인계 요약 생성
```

특정 날짜 범위:

```
오늘 수행한 작업의 인수인계 요약 생성
```

## 주요 섹션 설명

### 사용자 컨텍스트

대화 중 언급된 커뮤니케이션 선호도, 기술 수준, 특정 요구사항을 캡처한다

### 기술적 결정

특정 접근법을 선택한 이유를 문서화하여 향후 에이전트가 의도적인 선택을 되돌리는 것을 방지한다

### 코드 예시

복잡한 구현의 실제 코드 스니펫을 포함하여 다음 에이전트가 구현 스타일을 이해하도록 한다

### 중요 경고사항

알지 못하면 문제를 일으킬 수 있는 중요 정보를 강조한다 (예: "Y가 깨지므로 X를 수정하지 말아라")

### 관련 파일

생성되거나 수정된 모든 파일을 목적과 변경사항에 대한 간략한 설명과 함께 나열한다

## 다음 에이전트를 위한 참고사항

요약에 포함할 내용:

- 대화 톤과 스타일 선호도
- 세션 중 생성된 도구나 명령
- 사용자의 명시된 선호도 (예: 이모지 없음, 관례적 커밋 접두사 없음)
- 현재 작업 디렉터리와 환경 설정
- 언급된 외부 의존성이나 API 키

## 출력 옵션

명령이 수행하는 작업:

- 종합적인 마크다운 파일을 생성한다
- handoff-summary-[타임스탬프].md로 저장한다
- 검토를 위한 콘텐츠를 표시한다
- 선택적으로 중요한 결정에 대한 대화 기록 발췌를 포함한다

이를 통해 AI 에이전트 간 전환이나 휴식 후 작업 재개 시 완벽한 연속성을 보장한다.
